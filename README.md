# crypto_pipeline

Конвейер для получения и обработки криптовалютных данных из внешних API
Проект для автоматического сбора данных с криптовалютных сайтов-агрегаторов
В будущем будут добавлены BI-инструменты для просмотра статистики

В репозитории присутствуют файлы с тестами (например, `app/test_*.py`). Они сгенерированы с помощью LLM
Это факт именно тестовых файлов: я не скрываю использование LLM для их генерации
Весь остальной код (логика получения, обработка, пайплайн и т.д.) написан мной
